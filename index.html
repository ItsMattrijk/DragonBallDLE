❓ COMMENT JOUER - MODE CLASSIQUE

🎯 Objectif

Devinez le personnage de Dragon Ball du jour en utilisant les indices !

🎮 Comment jouer

  1. Tapez un nom de personnage dans la barre de recherche
  2. Sélectionnez le personnage dans les suggestions
  3. Observez les catégories qui s'affichent
  4. Utilisez les indices pour deviner le bon personnage

💡 Indices

Des indices se débloquent au fur et à mesure de vos essais pour vous aider !

⏰ Nouveau personnage

Un nouveau personnage à deviner apparaît chaque jour à minuit. Revenez demain pour un nouveau défi !

❓ COMMENT JOUER - MODE TECHNIQUE

🎯 Objectif

Devinez quel personnage utilise la technique affichée !

🎮 Comment jouer

  1. Une technique est affichée en haut
  2. Tapez un nom de personnage dans la barre de recherche
  3. Sélectionnez le personnage dans les suggestions
  4. ✅ = Bonne réponse, ❌ = Mauvaise réponse

💡 Indices

Débloquez des indices progressivement pour vous aider à trouver le personnage !

⏰ Nouvelle technique

Une nouvelle technique à deviner apparaît chaque jour à minuit !

❓ COMMENT JOUER - MODE MOITIE-MOITIE

🎯 Objectif

Devinez les deux personnages dont les images sont fusionnées côte à côte !

🎮 Comment jouer

  1. L'image fusionnée de deux personnages mystère s'affiche
  2. Choisissez si vous cherchez le personnage de gauche ou de droite
  3. Tapez un nom dans la barre de recherche
  4. ✅ = Bonne réponse, ❌ = Mauvaise réponse
  5. Trouvez les deux pour gagner !

💡 Déflou progressif

Plus vous faites d'essais, moins l'image est floue !

⏰ Nouvelle fusion

Une nouvelle paire de personnages apparaît chaque jour à minuit !

🔓 MODE ADMINISTRATEUR

⚠️ Vous êtes sur le point de régénérer les personnages !

Cela vous permettra de rejouer autant de fois que vous voulez aujourd'hui.
